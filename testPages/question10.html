Revision Question 10:

Create a while loop that would endlessly ask "continue? (y/n)".

If user enters:

"y" --> continue
"n" --> stops the loop

any other letter will result in a prompt("enter y or n only") followed by the same "continue?(y/n)" question.

Example Output:

continue?(y/n): y
continue?(y/n): y
continue?(y/n): t
enter y or n only
continue?(y/n): n
